# iot-tilt-monitor
Determining tilt angle of a 2-wheeler vehicle using MPU6050 sensor with Raspberry Pi Pico W

## Components
- Raspberry Pi Pico W
- Breadboard
- USB cable
- MPU6050 sensor
- Buzzer
- Connecting wires
- Toy bike

## Tools
- Thonny IDE

## Languages
- Micropython

## Modules
- mpu6050.py
- BlynkLib.py
